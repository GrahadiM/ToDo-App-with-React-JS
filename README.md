## Installation
### Clone Repo

```bash
git clone https://github.com/GrahadiM/ToDo-App-with-React-JS.git
```
### Go to folder

```bash
cd ToDo-App-with-React-JS
```
### Install and build App with npm

```bash
npm install
```
and
```bash
npm run dev
```
### Run App

```bash
npm run start
```
